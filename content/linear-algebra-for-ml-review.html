Week 1

Systems of Linear Equations

Singular & non-singular matrices · determinants · rank · linear dependence
🧠
Why ML cares

Neural networks are stacks of matrix operations

An image fed to a network is just a grid of numbers — a matrix. Each layer multiplies it by a weight matrix. Asking "does this picture contain a face?" reduces to repeated linear-algebra operations. Before you can train anything, you need to understand when those systems have answers.

From sentences to equations to lines

A system of equations is a set of conditions that share unknowns. The course builds intuition with "systems of sentences" (e.g. the dog is black, the cat is orange) and then translates them into algebra. The key insight carries over: a system can be complete (gives full information), redundant (repeats information), or contradictory (impossible).

Linear equation — every variable appears to the first power only; no products, powers, roots, or functions of the variables.
\[ a_1 x_1 + a_2 x_2 + \dots + a_n x_n = b \]

In two variables, a linear equation is a line; in three variables it is a plane. Solving a system geometrically means finding where those lines/planes intersect.

Two lines, three outcomes

  • Cross once → one solution → non-singular
  • Same line → infinitely many → singular (redundant)
  • Parallel → no solution → singular (contradictory)

System as a matrix equation

\[ \begin{bmatrix} a & b \\ c & d \end{bmatrix}\begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} e \\ f \end{bmatrix} \]

The coefficient matrix \(A\) holds the "structure"; the constants on the right don't affect whether \(A\) is singular.

Singular vs. non-singular matrices

A square matrix is non-singular if its rows (equivalently, its columns) are linearly independent — none can be built from the others. It is singular if there is linear dependence, which means the system carries less information than it appears to.

PropertyNon-singularSingular
Rows / columnsindependentdependent
Determinantdet ≠ 0det = 0
Solutions to \(Ax=b\)exactly onezero or infinitely many
Inverse \(A^{-1}\)existsdoes not exist
Rank (n×n)full (= n)less than n

A neat fact from the course: the constants on the right-hand side never change singularity — only the coefficient matrix does.

The determinant

The determinant is a single number computed from a square matrix that tells you, at a glance, whether it is singular. A matrix is singular exactly when its determinant is zero.

2 × 2 determinant

\[ \det\!\begin{bmatrix} a & b \\ c & d \end{bmatrix} = ad - bc \]

Product of the main diagonal minus product of the anti-diagonal.

3 × 3 determinant (Sarrus)

\[ \det\!\begin{bmatrix} a & b & c \\ d & e & f \\ g & h & i \end{bmatrix} = aei + bfg + cdh - ceg - bdi - afh \]

Add the three "down-right" diagonals, subtract the three "down-left" diagonals.

Worked 3×3 determinant, expanded line by line ▶

Find \(\det\begin{bmatrix}2&1&3\\1&0&2\\4&1&8\end{bmatrix}\) and decide if it is singular.

1
Expand along the first row using cofactors: \(\;\det = a(ei-fh) - b(di-fg) + c(dh-eg)\), with \(a=2,b=1,c=3\).
2
First minor: \(ei-fh = (0)(8)-(2)(1) = -2\). So the first term is \(2\cdot(-2) = -4\).
3
Second minor: \(di-fg = (1)(8)-(2)(4) = 0\). With the minus sign: \(-1\cdot 0 = 0\).
4
Third minor: \(dh-eg = (1)(1)-(0)(4) = 1\). So the third term is \(3\cdot 1 = 3\).
5
Add them: \(\det = -4 + 0 + 3 = \mathbf{-1}\). Since \(\det \neq 0\), the matrix is non-singular — its three rows are linearly independent.
Worked Is this 2×2 matrix singular? ▶

Test \(\begin{bmatrix}2&6\\3&9\end{bmatrix}\).

1
Compute the determinant: \(\det = ad-bc = (2)(9)-(6)(3) = 18 - 18 = 0\).
2
Because \(\det = 0\), the matrix is singular. Sanity check the dependence: row 2 \((3,9)\) is exactly \(1.5\times\) row 1 \((2,6)\), so the rows carry the same information.
3
Consequence: it has no inverse, and any system \(Ax=b\) built from it has either zero or infinitely many solutions — never a unique one.

Practice — Week 1

Q1Which matrix is singular?
Singular when det = 0. For the second matrix, \(\det = 2\cdot2 - 4\cdot1 = 0\); its second row is half the first, so the rows are linearly dependent.
Q2Two lines are parallel but distinct. The system is…
Parallel distinct lines never meet → no solution → contradictory. Both contradictory and redundant systems are singular (det = 0).
Q3\(\det\begin{bmatrix}3&1\\6&2\end{bmatrix}\) equals…
\(3\cdot2 - 1\cdot6 = 6-6 = 0\). The matrix is singular — row 2 is exactly twice row 1.
Q4Changing the constants \(c_1, c_2\) on the right-hand side of a system can change…
Singularity and the determinant depend only on the coefficient matrix. Constants shift the lines/planes around but don't change their slopes, so they can move where solutions are without affecting singularity.
Week 2

Solving Systems of Linear Equations

Elimination · row operations · rank · row echelon form & RREF
🗜️
Why ML cares

Image compression = reducing rank

A high-rank image matrix carries lots of independent information. Many compression methods deliberately lower the rank — keeping only the most informative directions — to shrink files while preserving what the eye notices. Rank measures how much genuinely independent information a matrix holds.

The elimination method

Solving a system means peeling away variables one at a time. You're allowed three reversible moves that never change the solution set:

① Swap
Exchange two rows (reorder equations).
② Scale
Multiply a row by a non-zero scalar.
③ Combine
Add a multiple of one row to another.

These are the elementary row operations. Repeatedly applying them is Gaussian elimination.

Watch for the singular cases

  • Redundant: a row becomes all zeros (\(0 = 0\)) → infinitely many solutions.
  • Contradictory: a row becomes \(0 = k\) with \(k\neq 0\) → no solutions.

Rank of a matrix

Rank — the number of linearly independent rows (equivalently, columns), i.e. the amount of genuinely independent information the system carries. It equals the number of pivots after row reduction.
n × n matrixRankStatus
All rows independent= n (full rank)non-singular
Some dependence< nsingular

Intuition from the course: rank counts the pieces of independent information. A 2×2 matrix whose second row repeats the first has rank 1 — only one real equation.

Row echelon form & RREF

Row Echelon Form (REF)

  • All-zero rows sit at the bottom.
  • Each leading entry (pivot) is to the right of the one above it — a staircase.
  • Entries below each pivot are zero.

Number of non-zero rows = rank.

Reduced Row Echelon Form (RREF)

  • Everything in REF, plus:
  • Every pivot equals 1.
  • Each pivot is the only non-zero entry in its column.

RREF reads off the solution directly — the goal of Gauss–Jordan elimination.

Worked Solve a 2×2 system by elimination, start to finish ▶

Solve \(\;2x + 3y = 8\;\) and \(\;x - y = -1\).

1
Write the augmented matrix: \(\left[\begin{array}{cc|c}2&3&8\\1&-1&-1\end{array}\right]\).
2
Swap rows so the pivot is a clean 1: \(\;R_1 \leftrightarrow R_2 \Rightarrow \left[\begin{array}{cc|c}1&-1&-1\\2&3&8\end{array}\right]\).
3
Eliminate \(x\) from row 2: \(\;R_2 \to R_2 - 2R_1 \Rightarrow \left[\begin{array}{cc|c}1&-1&-1\\0&5&10\end{array}\right]\).
4
Back-substitute: row 2 says \(5y = 10 \Rightarrow y = 2\). Row 1 says \(x - y = -1 \Rightarrow x = -1 + 2 = 1\).
5
Solution \((x,y) = (\mathbf{1},\mathbf{2})\). Two pivots ⇒ rank 2 ⇒ non-singular ⇒ unique solution. (Try these numbers in Tool 03 below to watch the steps.)

Practice — Week 2

Q5During elimination a row becomes \(0 = 5\). The system is…
\(0 = 5\) is impossible, so the system has no solution — it's contradictory and therefore singular.
Q6A 3×3 matrix row-reduces to two pivots and one zero row. Its rank is…
Rank = number of pivots (non-zero rows) = 2. Since rank < 3, the matrix is singular.
Q7Which is not a valid elementary row operation?
You may only scale by a non-zero scalar. Multiplying a row by 0 destroys information and would change the solution set.
Q8In RREF, every pivot must be…
RREF requires pivots = 1 and every other entry in a pivot column to be 0. That's the difference from plain REF.
Week 3

Vectors & Linear Transformations

norms · dot product · matrices as transformations · inverses · perceptrons
🤖
Why ML cares

A neuron is a dot product plus a decision

The perceptron — the atom of a neural network — takes inputs \(x\), computes the dot product with weights \(w\), adds a bias \(b\), and fires if the result clears a threshold. Words, images, and sentences all become vectors so models can measure similarity with the dot product.

Vectors, norms & direction

A vector is an arrow with magnitude and direction — or simply an ordered list of numbers, e.g. \((4,3)\). The norm measures its length.

L2 norm (Euclidean)

\[ \lVert v\rVert_2 = \sqrt{v_1^2 + v_2^2 + \dots + v_n^2} \]

Straight-line distance. \((4,3)\) has length \(\sqrt{16+9}=5\).

L1 norm (taxicab)

\[ \lVert v\rVert_1 = |v_1| + |v_2| + \dots + |v_n| \]

Distance walking along a grid. \((4,3)\) gives \(4+3=7\).

Vector operations

Sum
\((u_1+v_1,\;u_2+v_2)\) — tip-to-tail; the diagonal of the parallelogram.
Difference
\((u_1-v_1,\;u_2-v_2)\) — the arrow from \(v\)'s tip to \(u\)'s tip.
Scalar ×
\(\lambda v = (\lambda v_1, \lambda v_2)\) — stretches the vector; a negative \(\lambda\) flips it.

The dot product

The dot product turns two vectors into a single number and is the workhorse of ML similarity.

\[ u\cdot v = u_1 v_1 + u_2 v_2 + \dots + u_n v_n = \lVert u\rVert\,\lVert v\rVert\cos\theta \]

Norm from the dot product

\[ \lVert v\rVert = \sqrt{v\cdot v} \]

Orthogonality test

\[ u\cdot v = 0 \;\Longleftrightarrow\; u \perp v \]

A dot product of zero means the vectors meet at 90°.

A linear equation \(a_1x_1 + a_2x_2 = b\) is itself a dot product: \((a_1,a_2)\cdot(x_1,x_2)=b\).

Matrices as linear transformations

A matrix doesn't just store numbers — it moves space. Multiplying every point by a matrix \(A\) stretches, rotates, shears, or reflects the plane while keeping the origin fixed and grid lines straight and evenly spaced.

The columns of \(A\) tell you exactly where the basis vectors land:

\[ A = \begin{bmatrix} a & b \\ c & d \end{bmatrix} \;\Rightarrow\; \hat\imath \mapsto \begin{bmatrix} a \\ c \end{bmatrix},\quad \hat\jmath \mapsto \begin{bmatrix} b \\ d \end{bmatrix} \]

Knowing where \(\hat\imath\) and \(\hat\jmath\) go determines where everything goes.

Composing transformations = matrix multiplication

Doing transformation \(B\) then \(A\) is the single transformation \(AB\). This is why matrices multiply the way they do — and why order matters (\(AB \neq BA\) in general).

Identity matrix \(I\) — the "do nothing" transformation. Ones on the diagonal, zeros elsewhere; \(IA = AI = A\).

Matrix inverses

The inverse \(A^{-1}\) undoes what \(A\) does, so applying both returns you to the start:

\[ A\,A^{-1} = A^{-1}A = I \]

2 × 2 inverse formula

\[ A^{-1} = \frac{1}{ad-bc}\begin{bmatrix} d & -b \\ -c & a \end{bmatrix} \]

Swap the diagonal, negate the off-diagonal, divide by the determinant.

Which matrices have inverses?

Only non-singular ones. If \(\det A = 0\) you'd divide by zero — a singular transformation squashes space and cannot be reversed.

Worked Find and verify a 2×2 inverse ▶

Invert \(A=\begin{bmatrix}4&7\\2&6\end{bmatrix}\).

1
Determinant: \(\det = (4)(6)-(7)(2) = 24 - 14 = 10\). It is non-zero, so the inverse exists.
2
Apply the formula — swap the diagonal, negate the off-diagonal, divide by the determinant: \[ A^{-1} = \frac{1}{10}\begin{bmatrix}6&-7\\-2&4\end{bmatrix} \]
3
So \(A^{-1} = \begin{bmatrix}0.6&-0.7\\-0.2&0.4\end{bmatrix}\).
4
Verify \(AA^{-1}=I\): top-left entry \(= (4)(0.6)+(7)(-0.2) = 2.4 - 1.4 = 1\) ✓; top-right \(= (4)(-0.7)+(7)(0.4) = -2.8 + 2.8 = 0\) ✓. The product is the identity, as required.

Q9The L2 norm of \((6,8)\) is…
\(\sqrt{6^2+8^2}=\sqrt{36+64}=\sqrt{100}=10\).
Q10\((3,-2)\cdot(2,3)\) equals…
\(3\cdot2 + (-2)\cdot3 = 6-6 = 0\). The dot product is zero, so the two vectors are orthogonal.

The first column is the image of \(\hat\imath = (1,0)\) and the second column is the image of \(\hat\jmath = (0,1)\). That fully defines the transformation.

Week 4

Determinants & Eigenvectors

determinant as area · bases & span · eigenvalues/vectors · null space · PCA
📉
Why ML cares

PCA finds the directions that matter

Principal Component Analysis compresses high-dimensional data by keeping the directions of greatest variance. Those directions are the eigenvectors of the data's covariance matrix, ranked by their eigenvalues. Eigenvectors are how machines decide what to keep and what to throw away.

The determinant as an area

Geometrically, the determinant is the factor by which a transformation scales area (volume in 3D). The unit square has area 1; after applying \(A\), its area becomes \(|\det A|\).

\(\det = 2\)
areas double.
\(\det = 1\)
areas preserved (e.g. rotations, shears).
\(\det = 0\)
space collapses onto a line/point — singular.
\(\det < 0\)
area scales by \(|\det|\) and orientation flips (a reflection).

Determinant identities

\[ \det(AB) = \det(A)\det(B) \]

Compose two transformations, multiply their area factors.

\[ \det(A^{-1}) = \frac{1}{\det(A)} \]

Undoing a transformation inverts its area factor. \(\det(I)=1\).

Corollary: if either factor is singular, the product is singular — you can't un-squash space.

Bases, span & dimension

Span — every point you can reach by scaling and adding a set of vectors.
Basis — a minimal spanning set: enough vectors to reach everything, with none redundant (they must be linearly independent).
Dimension — the number of vectors in a basis.

Two independent vectors form a basis for the plane (dimension 2). If one is a multiple of the other, they only span a line — not a basis for the plane.

Eigenvalues & eigenvectors

Most vectors change direction under a transformation. A few special ones — the eigenvectors — stay on their own line; they only get stretched or squashed by a factor called the eigenvalue.

\[ A\,v = \lambda\,v \qquad (v \neq 0) \]

How to find them

Step 1 — eigenvalues. Solve the characteristic equation:

\[ \det(A - \lambda I) = 0 \]

For a 2×2 matrix this becomes a quadratic:

\[ \lambda^2 - (a+d)\,\lambda + (ad-bc) = 0 \]

The coefficients are the trace \((a+d)\) and the determinant \((ad-bc)\).

Step 2 — eigenvectors. For each \(\lambda\), solve \((A-\lambda I)v = 0\) to find the directions that survive. The eigenvectors form an eigenbasis in which the transformation is just stretching along axes.

Worked Find the eigenvalues and eigenvectors of a 2×2 matrix ▶

Take \(A=\begin{bmatrix}2&1\\1&2\end{bmatrix}\).

1
Build the characteristic equation \(\det(A-\lambda I)=0\): \[ \det\begin{bmatrix}2-\lambda&1\\1&2-\lambda\end{bmatrix} = (2-\lambda)^2 - 1 = 0 \]
2
Expand: \((2-\lambda)^2 = 1 \Rightarrow 2-\lambda = \pm 1\). So the eigenvalues are \(\lambda_1 = 3\) and \(\lambda_2 = 1\). Check: their sum \(3+1=4\) is the trace, and product \(3\cdot1=3\) is the determinant. ✓
3
Eigenvector for \(\lambda_1=3\): solve \((A-3I)v=0 \Rightarrow \begin{bmatrix}-1&1\\1&-1\end{bmatrix}v=0\), giving \(-v_1+v_2=0\), i.e. \(v_1=v_2\). Eigenvector \(v_1 = (1,1)\).
4
Eigenvector for \(\lambda_2=1\): solve \((A-I)v=0 \Rightarrow \begin{bmatrix}1&1\\1&1\end{bmatrix}v=0\), giving \(v_1+v_2=0\). Eigenvector \(v_2 = (1,-1)\).
5
Interpretation: along \((1,1)\) the matrix stretches by \(\times 3\); along the perpendicular \((1,-1)\) it leaves length unchanged \((\times 1)\). Enter this matrix in Tool 06 to see those two lines drawn.

Null space & orthogonal matrices

Null space

\[ \text{Null}(A) = \{\,v : A v = 0\,\} \]

The set of vectors that \(A\) sends to the origin. A non-singular matrix has only \(\{0\}\) in its null space; a singular matrix has a whole line or plane of solutions — which is why singular systems have infinitely many solutions.

Orthogonal matrix

\[ Q^{\mathsf T} Q = I \]

Columns are mutually orthogonal unit vectors. \(Q\) preserves lengths and angles — it only rotates or reflects, never stretches. Hugely useful in PCA and numerically stable algorithms.

Putting it together — PCA

Principal Component Analysis is where the whole course converges:

  1. Center the data and build the covariance matrix (symmetric).
  2. Find its eigenvectors (the principal directions) and eigenvalues (variance along each).
  3. Keep the top few eigenvectors — the directions of greatest spread — and project the data onto them.

Determinants told you when this is possible, dot products measure the projections, and eigenvectors pick the directions worth keeping. That's linear algebra doing dimensionality reduction.

Practice — Week 4

Q13A transformation has \(\det = 0\). Geometrically it…
\(\det = 0\) means the area-scaling factor is zero — the transformation flattens 2D space, so it's singular and non-invertible.
Q14If \(\det A = 3\) and \(\det B = 4\), then \(\det(AB)\) is…
\(\det(AB)=\det(A)\det(B)=3\times4=12\).
Q15For \(A=\begin{bmatrix}2&0\\0&3\end{bmatrix}\), the eigenvalues are…
For a diagonal matrix the eigenvalues are simply the diagonal entries: 2 and 3, with eigenvectors along the x- and y-axes.
Q16In the characteristic equation \(\lambda^2-(a+d)\lambda+(ad-bc)=0\), the constant term is the…
The \(\lambda\)-coefficient is the trace \((a+d)\); the constant term is the determinant \((ad-bc)\). So the product of the eigenvalues equals the determinant.
Q17PCA's principal components are the…
PCA keeps the top eigenvectors of the covariance matrix — the directions of greatest variance, ranked by eigenvalue.
Q18A non-singular matrix's null space contains…
Only \(\{0\}\). If a non-zero vector were sent to the origin, the transformation would be collapsing space and would be singular.

Quick recall — reveal the answers

📝 In one sentence: what is an eigenvector, and what does its eigenvalue tell you?
An eigenvector is a non-zero vector whose direction is unchanged by the transformation; its eigenvalue \(\lambda\) is the factor by which it is stretched (\(>1\)), shrunk (\(0<\lambda<1\)), or flipped (\(\lambda<0\)). Formally \(Av=\lambda v\).
📝 Give three equivalent statements that mean a square matrix \(A\) is non-singular.
\(\det A \neq 0\); the rows (and columns) are linearly independent; \(A\) has full rank; \(A^{-1}\) exists; \(Ax=b\) has exactly one solution; the null space is just \(\{0\}\). (Any three.)
